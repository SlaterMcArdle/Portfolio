About Me

Full stack web developer leveraging a diverse physics and engineering background to create a seamless, functional user interface. Currently studying MERN full stack development at UC San Diego. This is coupled with my previous experience developing web apps using Django. Working in a wide variety of industries from maritime to retail to a variety of robotics, I’ve learned what it takes to create a robust product that is intuitive and user friendly. With each project, my aim is to create an intuitive, reliable user experience. I worked with a team of three to develop a single-page frontend web application that provides users the ability to find and navigate to the nearest available restroom. I also worked with a team of two developing a django remote web application to easily control remote fluid dispensing via a linux-based microcomputer.
